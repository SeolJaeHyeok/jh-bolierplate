# jh-bolierplate

Simple boilerplate with [React](https://github.com/facebook/react), [TypeScript](https://github.com/microsoft/TypeScript)

## Dependencies

- react
- react-dom
- react-router-dom
- eslint
- prettier
- crossenv

## Quick Overview

```bash
npx create-jh-app my-app
cd my-app
npm run dev
```
